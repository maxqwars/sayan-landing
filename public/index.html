Открытое приложение для работы с REST API Анилибрии


Что такое Sayan?

Саян приложение с открытым исходным кодом, для работы с API Анилибрии (или иным совместимым с ним). Позволяет просматривать информацию о релизе, искать релизы в базе данных и просматривать прикрепленный к релизу контент (видео плеер, торрент). Саян доступен как веб приложение и как нативное приложение для вашей операционной системы.

Особенности

code

Открытый

Исходный код Саян(а) распространяется в соответствии с публичной лицензией GNU. Эта лицензия дает вам право исследовать, модифицировать и распространять исходный код Саян(а) или ваших модификаций, при условии соблюдения лицензии GNU.

accessibility

Маштабируемый

Саян использует веб технологии, чтобы обеспечить высокую переносимость основных функций, а так же набор расширений для более тесной интеграции с системой или реализации дополнительной функциональности.

flare

Современный

Саян написан на TypeScript, с использованием ReactJS и применением компонентов Ionic Framework. Саян использует возможности современных браузеров, чтобы предоставить лучший опыт использования веб приложений.
